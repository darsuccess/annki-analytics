## E. ДИВАН PLUS -- ДЕТАЛЬНЫЙ РАСЧЕТ

### Конфигуратор дивана Plus

Диван Plus -- модульный. Каждый модуль может быть шириной от 50 до 120 см. Стандартный диван с посадкой 200 см: **2 раскладных модуля + 2 подлокотника (локтя)**.

**Базовые модули (без ткани):**
- 2 раскладных модуля: 24,500 x 2 = 49,000
- 2 локтя (подлокотника): 8,300 x 2 = 16,600
- **Итого база: 65,600**

Расход ткани: 2 модуля x 8 мп + 2 локтя x 2.5 мп = 21 мп

**Более точный расчет на основе данных из таблицы себестоимости:**

Из себестоимости раскладного модуля и 2 локтей:
- Модуль раскл. (всего с работой): 24,500 + ткань (8 мп) -- x2
- Локоть (всего с работой): 8,300 + ткань (2.5 мп) -- x2

Итого для базовой рогожки (350 руб/мп):
- 2 модуля: 2 x (24,500 + 8 x 350) = 2 x 27,300 = **54,600**
- 2 локтя: 2 x (8,300 + 2.5 x 350) = 2 x 9,175 = **18,350**
- **ИТОГО: 72,950**
